    FIDEL Student Portal System

    => The software is a School Portal System which is a versatile system which consists of the basic utilities that any school needs.

    => All entities of schools (administrators, teachers and students) will be capable of using this system based on their need and level of access permission.

    => The system allows teachers to submit students' grades, access basic students and parents information, and share resources for their students and communicate with them.

    => It allows students to see their results, communicate with their teachers and access shared resources.

    => Administrators are the one who control the whole events going on in the school starting from registering students and hiring teachers to telling announcements.

    => The system will have a database that contains users detail, resources and other data essential for the school system

    => The system is not concerned with school fees, payments and any type of transactions done under the school.

    => Although the system is used to share resources it does not provide library looking services like renting books or other services given in a library.
